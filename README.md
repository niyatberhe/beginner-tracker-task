# Simple Task Tracker

A beginner-friendly task tracker built with HTML, CSS, and JavaScript. Perfect for learning web development basics!

**[Live Demo](https://beginner-tracker-task.vercel.app/)** ← here

## Features

- ✅ Add new tasks
- ✅ Mark tasks as complete/incomplete
- ✅ Delete tasks
- ✅ Filter tasks (All, Active, Completed)
- ✅ Clear all completed tasks
- ✅ Task counter
- ✅ Local storage persistence
- ✅ Responsive design
- ✅ Beautiful gradient UI

## How to Use

1. **Add a task**: Type your task in the input field and click "Add Task" or press Enter
2. **Complete a task**: Click the checkbox next to a task
3. **Delete a task**: Click the "Delete" button
4. **Filter tasks**: Use the filter buttons to view All, Active, or Completed tasks
5. **Clear completed**: Click "Clear Completed" to remove all finished tasks

## Code Explanation for Beginners

### HTML (`index.html`)
- **Semantic structure**: Uses `<header>`, `<main>`, `<section>` tags
- **Form handling**: Input field with form validation
- **Interactive elements**: Buttons with data attributes for filtering

### CSS (`styles.css`)
- **Modern styling**: Flexbox layout, gradients, transitions
- **Responsive design**: Media queries for mobile devices
- **Component-based**: Organized by sections (header, filters, tasks)

### JavaScript (`script.js`)
- **Class-based architecture**: `TaskTracker` class organizes all functionality
- **Event handling**: Listens for form submissions, button clicks
- **Local storage**: Saves tasks between browser sessions
- **DOM manipulation**: Dynamic rendering of tasks
- **State management**: Tracks current filter and task list
