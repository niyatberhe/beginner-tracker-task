# Task Tracker - Hackathon Project

## 🚀 Project Overview
A simple, elegant task tracker built for beginners. This project demonstrates core web development skills with a clean, functional design.

## 🎯 Target Audience
- Beginner developers learning HTML, CSS, JavaScript
- Hackathon participants looking for a clean, functional project
- Anyone wanting a simple productivity tool

## ✨ Key Features
- **Task Management**: Add, complete, and delete tasks
- **Smart Filtering**: View All, Active, or Completed tasks
- **Data Persistence**: Tasks save automatically using localStorage
- **Responsive Design**: Works perfectly on desktop and mobile
- **Modern UI**: Beautiful gradient design with smooth animations
- **Task Statistics**: Real-time task counter
- **Bulk Actions**: Clear all completed tasks at once

## 🛠️ Technologies Used
- **HTML5**: Semantic structure and accessibility
- **CSS3**: Modern styling with Flexbox, transitions, gradients
- **JavaScript ES6+**: Classes, arrow functions, localStorage API
- **No frameworks** - Pure vanilla web technologies
